# CodeAlpha_Task1_Language_Translation_Tool

# Task 1: Language Translation Tool

This task was developed as part of the CodeAlpha AI/ML Internship and implemented on Google Colab.

## Description

This is a Python application that translates text between 100+ languages using the Google Translate API.

### Features

* Real-time text translation between 100+ languages
* Terminal-based input (originally with a GUI using ipywidgets in Google Colab)

### Libraries Used

* `googletrans==4.0.0-rc1` (Google Translate API)
* `httpx` (for network requests)
* `nltk` (Natural Language Toolkit)

### Setup & Installation

All required libraries can be installed with a single command:

```
pip install googletrans==4.0.0-rc1 httpx
```

### How to Run

1. Open the Google Colab notebook: [Link to Colab Notebook] (https://colab.research.google.com/drive/1NA3Mw6MFm9uLXS1CVWPBUzuCCYT84E7Z?usp=sharing).
2. Follow the steps in the notebook: Enter the text, select source and target language codes, and run the cells.
3. The translated text will be displayed in real-time.
